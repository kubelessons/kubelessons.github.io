# MiniKube Setup

## Description
MiniKube is a tool that makes it easy to run Kubernetes locally. It runs a by default single-node Kubernetes cluster inside a VM on your computer.

## Prerequisites
- Linux

## Installation Steps

### 1. Install MiniKube
```bash
curl -LO https://storage.googleapis.com/minikube/releases/latest/minikube-linux-amd64
sudo install minikube-linux-amd64 /usr/local/bin/minikube
